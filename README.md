# 💕 Моя Любимая - Интерактивная Книжка

Красивая интерактивная книжка-поздравление с Днем Рождения!

## 📁 Структура проекта

```
project/
├── index.html          # Главный файл (открыть в браузере)
├── styles/             # CSS файлы для стилей
│   ├── variables.css       # CSS переменные и цвета
│   ├── reset.css           # Нормализация элементов
│   ├── background.css      # Фоновые элементы
│   ├── book.css            # Основная структура книги
│   ├── pages.css           # Стили страниц
│   ├── typography.css      # Шрифты и текст
│   ├── photos.css          # Фотографии и загрузка
│   ├── decorations.css     # Анимации и эффекты
│   ├── navigation.css      # Кнопки навигации
│   └── responsive.css      # Адаптивность
└── js/
    └── main.js         # Вся логика приложения
```

## 🚀 Как использовать

### Быстрый старт

1. **Скачай все файлы** - сохрани папку проекта себе на компьютер
2. **Добавь свои фото и видео** - положи файлы в папку `photos/` с именами:
   - `photo1.jpg` - для страницы 2 (вас вместе)
   - `photo2.jpg` - для страницы 3 (ваш момент)  
   - `video3.mp4` или `video3.webm` - для страницы 5 (воспоминание в видео)
3. **Открой `index.html`** - просто дважды кликни на файл (откроется в браузере)
4. **Листай страницы** - используй кнопки "Назад" и "Вперед"

### Песня

Добавь файл (в папку `photos/`):
- `song.mp3` или `song.ogg` - для страницы 6 (песня)

### Фотографии и видео

Файлы должны быть в папке `photos/`:
- **photo1.jpg** - показывается на странице 2
- **photo2.jpg** - показывается на странице 3
- **video3.mp4** или **video3.webm** - показывается на странице 5 (видео с автозапуском)
- **song.mp3** или **song.ogg** - показывается на странице 6 (песня)

Видео автоматически начинает воспроизводиться при переходе на страницу 5 и останавливается при уходе с неё. Если браузер блокирует автозапуск, появятся элементы управления.

Если файл не найден, будет показано сообщение "Фото не найдено" или "Видео не найдено".

### Управление

- **Мышка** - кнопки "Назад" и "Вперед"
- **Клавиатура** - стрелки ← → или пробел
- **Сенсор** - свайп влево/вправо
- **Home/End** - в начало/конец книги

## 🛠 Кастомизация

### Изменить текст

1. Открой `index.html` в текстовом редакторе (VS Code, Notepad++)
2. Найди нужный текст внутри тегов `<p>` или `<h2>`
3. Измени текст как хочешь
4. Сохрани файл
5. Обнови страницу в браузере (F5)

### Изменить цвета

1. Открой `styles/variables.css`
2. Найди раздел `Primary Colors`
3. Измени значения цветов (hex коды: #ff4d7d, #ff8fa3 и т.д.)
4. Сохрани файл

Примеры цветов:
- `#ff4d7d` - основной розовый
- `#ff8fa3` - светлый розовый
- `#4d7fff` - синий
- `#ff4d8f` - малиновый

## 💾 Сохранение фотографий

Фотки автоматически сохраняются в памяти браузера (localStorage).
Если закроешь и заново откроешь - фотки останутся!

⚠️ **Внимание**: Если очистить кэш браузера - фотки удалятся.

## 📊 Советы

- **Фотографии лучше загружать в формате JPG или PNG**
- **Размер файла максимум 5MB**
- **Лучше использовать квадратные или близкие к квадрату фотографии**
- **Текст автоматически будет красиво выглядеть на любом устройстве**

## 🌐 Отправка подруге

Есть несколько способов:

### Способ 1: Отправить файл
Сожми папку проекта (правой кнопкой → сжать) и отправь ей файл `.zip`
Она распакует и откроет `index.html`

### Способ 2: Загрузить на веб-хостинг
Загрузи файлы на бесплатный хостинг типа:
- Netlify (netlify.com)
- Vercel (vercel.com)
- GitHub Pages

И отправь ей ссылку

### Способ 3: Просто открыть у неё
Открой файл `index.html` на её компьютере и дай ей полюбоваться 💕

## 🐛 Решение проблем

**Фотки не загружаются:**
- Проверь расширение файла (jpg, png, gif)
- Размер файла не больше 5MB
- Попробуй перезагрузить страницу (F5)

**Стили не применяются:**
- Убедись что папка `styles/` находится в той же папке что и `index.html`
- Проверь что все файлы на месте
- Попробуй обновить кэш (Ctrl+Shift+Delete → очистить кэш)

**Скрипты не работают:**
- Убедись что папка `js/` находится в той же папке что и `index.html`
- Открой DevTools (F12) и посмотри на консоль на ошибки
